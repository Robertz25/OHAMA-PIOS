# OHAMA-PIOS
For school project in the IS-213 Opensource software


Hardware:
  * Raspberry Pi 3
  * Dust particle sensor (more info to come)
  * Humidity Sensor
  * Temperature sensor
  * Sound sensor (dB monitoring)
  * Brightness sensor
  * Hazardous Gas 

Software: 
  * Python 3.7
  * OpenMCT
  * Docker container
  * Node.JS
